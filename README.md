# Sallus Flow — Site institucional

Site moderno e interativo para `www.sallusflow.com.br`, alinhado ao visual do sistema Sallus/IMEC Instrument.

## Stack

- Vite
- React
- TypeScript
- CSS puro com tokens visuais do sistema
- Pronto para Vercel

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy na Vercel

1. Suba esta pasta para um repositório no GitHub.
2. Na Vercel, clique em **Add New Project**.
3. Selecione o repositório.
4. Framework Preset: **Vite**.
5. Build Command: `npm run build`.
6. Output Directory: `dist`.
7. Configure o domínio `www.sallusflow.com.br`.
8. Aponte o domínio raiz para redirecionar para `www`.
